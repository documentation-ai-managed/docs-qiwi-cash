## Authentication Process

To use Qiwi Cash API, include your API key in the `Authorization` header.

## Steps

1. Login to `https://cabinet.qiwi.cash`.
2. Go to **API Keys** in settings.
3. Create a new API key and copy it securely.

## Example Usage

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://api.qiwi.cash/v1/balance
```
